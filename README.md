# 📚 StudyMate — Saath Padho, Saath Badho

> Find your study peers, join live study rooms & clear doubts together.  
> A platform built for Indian students. 🇮🇳

## 💡 Problem

India mein lakho students akele padhte hain.  
- Doubt aata hai — koi nahi hota poochne ke liye  
- Motivation khatam hoti hai — saath padhne wala koi nahi  
- WhatsApp groups mein sab off-topic ho jaate hain  

**StudyMate iska solution hai.**

---

## ✨ Features

| Feature | Description |
|--------|-------------|
| 🔍 **Peer Finder** | Same subject, same exam wale students dhundo |
| 📺 **Live Study Rooms** | Camera on karo, saath padho virtually |
| 💬 **Doubt Chat** | Photo kheechi, subject-wise rooms mein doubt poocho |
| 🎯 **Study Streams** | Live stream shuru karo, doosre join karein |

---

## 🎯 Target Users

- JEE / NEET Aspirants
- Class 10 / 12 Board Students (CBSE, RBSE, ICSE)
- UPSC / SSC Prep Students
- College Students
- Coding Seekhne Wale
